## 🍌 Nano Banana Pro 提示詞

**Nano Banana Pro** 是 Google 最新的多模態 AI 模型，具有以下特點：

- 🎯 **多模態理解** - 處理文本、圖像和視頻
- 🎨 **高質量生成** - 從照片級真實感到藝術風格
- ⚡ **快速迭代** - 快速編輯和變化
- 🌈 **多樣風格** - 從像素藝術到油畫
- 🔧 **精確控制** - 詳細的構圖和光照
- 📐 **複雜場景** - 多對象、多角色渲染

- 個人檔案 / 頭像

- 資訊圖表 / 教育視覺化內容
- YouTube 縮圖
- 漫畫 / 分鏡腳本
- 產品行銷
- 電子商務主圖
- 遊戲素材
- 海報／傳單
- App / 網站設計

**風格：**
- 攝影 / 電影感
- 動畫 / 漫畫
- 插圖 / 草圖
- 3D 渲染 / 等距視角
- 像素藝術 / Q 版風格
- 油畫 / 水彩 / 水墨
- 復古 / 賽博龐克

## ♊ Gemini 3 提示詞

**Gemini 3** 是 Google 發布的新一代多模態大模型，具有以下特點：

- 🎯 **超大上下文窗口** - 支持 100 萬 tokens，全面理解上下文
- 🎨 **多模態智能** - 理解並生成文本、圖像、音頻、視頻、代碼等多種輸入
- ⚡ **執行型 AI** - 自動化執行，可直接生成網站並實時預覽，一鍵發布
- 🌈 **強大推理能力** - 高級問題解決和複雜任務處理

## 🚀 Raycast 集成

部分提示詞支持使用 [Raycast Snippets](https://raycast.com/help/snippets) 語法的**動態參數**。尋找 🚀 Raycast Friendly 徽章！

**示例：**
```
A quote card with "{argument name="quote" default="Stay hungry, stay foolish"}"
by {argument name="author" default="Steve Jobs"}
```

在 Raycast 中使用時，您可以動態替換參數以快速迭代！
